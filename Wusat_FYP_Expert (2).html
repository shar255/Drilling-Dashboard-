HOW LIVE RHEOLOGY WORKS
1 · Source of values
If you have entered lab data for the current NP system and concentration in the Lab Data tab, those actual measured values are shown here. If no lab data exists, values are calculated from your sidebar inputs using Bingham Plastic equations. The mode badge shows which is active.
2 · The trend charts
Every 2 seconds a new reading is added. If you change a parameter (e.g. raise KCl or switch NP type), the next reading reflects that change immediately. The chart builds up a 30-point rolling history so you can see how the mud responds as you adjust inputs.
3 · The Bingham flow curve
The rheogram plots shear stress (Pa) against shear rate (s⁻¹) at 6 viscometer speeds: R3, R6, R100, R200, R300, R600. The slope of the line is PV. The Y-intercept is YP. The green dashed line is your target (Hybrid 0.50 wt%). Your system should sit close to it.
4 · For the interview
This tab simulates what you would monitor in real time at the rig site if a Fann VG35 viscometer was connected to this platform. A drilling engineer reads R600 and R300 every 20 minutes. Those two readings give PV and YP which tell whether the mud is thickening or thinning as it circulates through swelling shale.
